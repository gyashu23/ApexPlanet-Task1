# ApexPlanet-Task1
A simple smartwatch showcase website created as part of the ApexPlanet Web Development Internship Task 1.

# Project Overview

This project is a simple smartwatch showcase website built using HTML, CSS, and JavaScript.

# Features

- Responsive smartwatch product cards
- Navigation bar
- Product images and descriptions
- Buy Now and Add to Cart buttons
- JavaScript alert messages for button clicks
- Clean and beginner-friendly UI

# Technologies Used

- HTML5
- CSS3
- JavaScript
